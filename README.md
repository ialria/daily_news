Pakistani News Web App

A simple and responsive **News Website** built using **HTML, CSS, JavaScript, and Bootstrap** that fetches and displays the latest Pakistani news in real-time using an API.

Features

 Live News Fetching
- Fetches real-time Pakistani news using an external API  
- Displays latest headlines dynamically  

 Loading Animation
- Shows a smooth loader while fetching data  
- Improves user experience  

 Responsive Design
- Built with Bootstrap  
- Works on mobile, tablet, and desktop  

News Cards UI
Each news item includes:
- Image  
- Title  
- Publish date  
- Clickable link to full article  

 How It Works

1. Page loads  
2. Loader appears  
3. API request is sent using `fetch()`  
4. Data is received in JSON format  
5. News items are dynamically rendered on screen  
6. Loader disappears  

 Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Bootstrap 5  
- Fetch API  

 How to Run

1. Download or clone the project  
2. Open `index.html` in your browser  

No installation required 🚀
